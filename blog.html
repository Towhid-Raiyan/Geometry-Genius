1. What is DOM? What is the purpose of DOM?

The Document Object Model in short term DOM, is a programming interface for web documents. It defines a standard for accessing documments. By DOM we can access documents, can update content, structure or style of a document.

2. How will you select HTML elements using DOM? Name the DOM methods.

By 5 ways we can select HTML elements using DOM. Name of the methods given below:
i. getElementsByTagName()
ii. getElementsByClassName()
iii. getElementById()
iv. querySelector()
v. querySelectorAll()

3. What is event bubble?

Event Bubble is an important element of DOM. In a event bubble system it's direct an event to its target. When we click an element, an event directed to its element. Then if a particular event handler set from before for that parrticular element, then event handler get triggered. Then the event bubble up from the elements to its parent till root.

4. What is a callback function and why will you use it?

When we send an individual function as a parameter of another particular function we call it a callback function. In this case the function which we called will be executed first and then will execute the function which we sent as a parameter.
